The javascript dependencies are managed by a nx monorepo.

**Run scripts**

In the root folder

```powershell
npm i
```

Build all modules

```powershell
npx nx run-many -t build --verbose
```

Test all modules

```powershell
npx nx run-many -t test --stream
```

Building a sepecific module

```powershell
npx nx build @morgan-stanley/composeui-messaging-client OR nx run @morgan-stanley/composeui-messaging-client:build
```

Visualize all modules in the workspace
```powershell
npx nx graph
```
